# Reorganizador de Leads (Python)

Ferramenta que lê uma planilha de leads (CSV ou Excel), detecta as colunas e permite ao usuário reorganizar, renomear e limpar os dados, gerando uma nova planilha organizada.

Funcionalidades
- Renomear colunas
- Reordenar colunas
- Remover colunas desnecessárias
- Formatar numeros no padrão brasileiro
- Exportar em Excel

## Tecnologias
- Python

## Como executar
```bash
python interface.py
